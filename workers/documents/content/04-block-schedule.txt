BLOCK SCHEDULE SYSTEM
=====================

A time-blocking feature that helps users plan their day with structured work sessions.

Overview
--------
The Block Schedule allows users to pre-plan their workday with dedicated focus blocks. This follows productivity methodologies like Cal Newport's "Deep Work" and the Pomodoro Technique.

Block Templates
---------------

1. STANDARD WORKDAY
   - 4 focus blocks with short breaks
   - Classic 8-hour structure
   - 480 minutes work, 60 minutes break
   - Best for: Traditional work schedules

2. POMODORO EXTENDED
   - 8 pomodoro sessions
   - 25 minutes work, 5 minutes break each
   - 200 minutes work, 65 minutes break
   - Best for: Tasks requiring frequent mental refresh

3. DEEP WORK DAY
   - 3 long focus sessions
   - Designed for complex, creative work
   - 480 minutes work, 90 minutes break
   - Best for: Programming, writing, design

4. FLEXIBLE HALF-DAY
   - 4-hour flexible schedule
   - 240 minutes work, 30 minutes break
   - Best for: Part-time workers or focused mornings

Features
--------

VISUAL TIMELINE
- Horizontal timeline showing all blocks
- Color-coded by type (work/break)
- Current block highlighted
- Progress indicators

BLOCK COMPLETION
- Mark blocks as complete
- Earn XP for completing blocks
- Milestone rewards for full day completion

CARRY-OVER SYSTEM
- Tracks incomplete blocks
- Carries deficit to next day
- Helps maintain consistency
- Can be manually cleared

CUSTOMIZATION
- Adjustable start time
- Template selection
- Custom block creation (future)

Gamification Integration
------------------------
- XP awarded per block completion
- Bonus XP for full day completion
- Streak tracking for consecutive days
- Integration with main gamification system

Data Storage
------------
- Blocks stored in localStorage for privacy
- Carry-over persisted between sessions
- Template preferences saved
- Stats tracked for analytics
