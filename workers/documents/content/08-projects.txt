PROJECT MANAGEMENT
==================

Organize work by projects with color-coding and time tracking integration.

Overview
--------
Projects allow users to categorize their time entries, making it easy to track hours spent on different clients, tasks, or initiatives.

Project Properties
------------------
- Name: Required, descriptive title
- Description: Optional details
- Color: Visual identifier (8 preset colors)
- Active: Boolean status (active/archived)

Color Options
-------------
- Relic Gold (#cca43b) - Default
- Hologram Cyan (#00f0ff)
- Baked Clay (#9f5f3f)
- Emerald (#046c4e)
- Sapphire (#0f5298)
- Ruby (#9b111e)
- Amethyst (#7c3aed)
- Coral (#f97316)

Admin Features
--------------
Only admin users can:
- Create new projects
- Edit existing projects
- Archive projects (soft delete)
- Change project colors
- Update descriptions

All users can:
- View project list
- Assign time to projects
- Filter time entries by project

Project Cards
-------------
Each project displays:
- Color indicator dot
- Active/Archived badge
- Project name
- Description snippet
- Project ID
- Edit button (admin only)

Time Integration
----------------
- Select project when clocking in
- Change project during session
- Project breakdown in analytics
- Hours per project in reports

API Endpoints
-------------
GET /api/projects
  - List all projects
  - Includes active and archived

POST /api/projects (admin only)
  - Create new project
  - Required: name
  - Optional: description, color

PUT /api/projects/:id (admin only)
  - Update project details
  - Can change any property

DELETE /api/projects/:id (admin only)
  - Soft delete (sets active = false)
  - Preserves historical data

Data Model
----------
projects:
  - id: Primary key
  - name: Text (required)
  - description: Text (optional)
  - color: Hex color string
  - active: Boolean (default true)
  - created_at: DateTime
  - updated_at: DateTime
