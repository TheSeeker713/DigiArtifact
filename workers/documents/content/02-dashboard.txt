DASHBOARD - COMMAND CENTER
===========================

The Dashboard is the central hub where users land after logging in. It provides an at-a-glance view of the current work status and quick access to all major features.

Main Sections
-------------

1. WELCOME HEADER
   - Personalized greeting with user's first name
   - Current date display with customizable format
   - Sets a professional yet friendly tone

2. QUICK STATS BAR
   - Today's total hours worked
   - Weekly hours accumulated
   - Active project count
   - Current streak (consecutive days worked)

3. CLOCK WIDGET (Primary Action)
   - Large, prominent clock in/out button
   - Real-time session timer when clocked in
   - Break timer with pause/resume
   - Session notes field for context
   - Project selector dropdown

4. FOCUS TIMER
   - Pomodoro-style focus sessions
   - Configurable work/break intervals
   - Audio notifications (optional)
   - Session history tracking

5. BODY DOUBLING TIMER
   - Virtual co-working feature
   - Designed for remote workers who benefit from "presence"
   - Ambient timer with encouraging messages
   - Helps with accountability

6. TODAY'S AGENDA
   - Scheduled blocks for the day
   - Quick view of planned work sessions
   - Links to Block Schedule for editing

7. STREAK COUNTER
   - Visual streak display with fire icon
   - Motivational messaging
   - Record streak tracking

8. QUICK NOTES WIDGET
   - Rapid note capture
   - Auto-saves to Journal
   - Tagged with source "quick_note"

9. GAMIFICATION WIDGET
   - Current level and XP progress
   - Recent achievements
   - Leaderboard position (if enabled)

10. WEEKLY CHART
    - Bar chart showing hours per day
    - Visual pattern recognition
    - Trend indicator

11. RECENT ENTRIES
    - Last 5 time entries
    - Quick edit access
    - Project color coding

Layout Philosophy
-----------------
- Three-column responsive grid
- Most important actions on the left
- Progress/stats on the right
- Mobile-first design collapses to single column
